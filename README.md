# Desafio Front_End - Tela de Login

## Instalação do Projeto

- Para instalar as dependências do projeto, execute o seguinte comando:

```
npm install
```
## Como rodar o Projeto

- Para rodar o projeto neste modo, utilize o comando abaixo:

```
npm run dev
```
